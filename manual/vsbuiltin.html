    Visual Scripting ( Unity >= 2021.1 )

    First of all, go to Unity->Preferences->Surfer , select Visual Scripting under the section Integrations and then click Save And Recompile.

    For 2021.1 or newer, Surfer has 11 Custom Units that can be used : 4 Events and 7 Actions

    Actions

    All the Unit Actions can be found under the /Surfer folder of the Unit's search menu.
