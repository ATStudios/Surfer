    Back Button

    Let’s setup a “Back button”, an action that will happen without clicking a specific UI element and only when that state is open.

    To do this, the event to use is :

    • Input/OldInput/OnKeyDown for the Old Input System
    • Input/NewInput/OnAction for the New Input System
    • Input/Rewired/OnAction for Rewired

    Next, let's choose the input key that will trigger the action.

    Finally, in the If...section ( category Fast! ) let's choose State/IsMyStateOpen : this condition will make sure that the input action will be performed only when the state of the object where the component SUElement has been added on, is open; without this condition the input action will be always triggered.
