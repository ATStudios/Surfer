    Theme Manager

    (This guide is only related to Canvas/UGUI)

    To setup our themes, let's go to Unity->Preferences->Surfer->Theme Manager.

    Then, let's create a new theme called redTheme and watch our window populate.



    Now , let's add a new color by clicking + next to the same category, and let's choose Red as an example color from the picker.



    Then, let's create a new theme called blueTheme and, from the list of properties, let's Edit the color of our my color to Blue.




    Now, in our scene, let's create a simple UI Canvas Button, let's add the SUThemable component to it and finally, let's choose my color under the color category. Then, hit play.


    Our button will be colored of blue cause our latest selected theme was blueTheme. So , let's exit PlayMode and go back to our Theme Manager Settings in the Preferences window and let's select redTheme from the dropdown list now.
    Then, hit play and our button will be now colored of red.




    Runtime

    To change theme at runtime we must simply call :

    SUThemeManager.I.ChangeTheme(_myThemeName);


    Instead, if we want to be notified when this event occurs, we must do as follows :

    public class MyScript : MonoBehaviour, ISUThemeChangeHandler
{


    void Awake() {
        SUThemeManager.I.RegisterThemeChange(this);
    }

    public void OnThemeChanged(SUThemeChangeEventData evtData)
    {
        //do my stuff
    }

}
