    Safe Area

    (This feature and guide are currently only available for Canvas/UGUI)

    Let's see how to automatically manage the Safe Area on a UIPanel.

    Runtime/Editor (or Device Simulator)

    Basically :

    • Add a Canvas (only for example purposes)
    • Add an empty UIPanel object with a Button inside , completely aligned to the left (only for example purposes)

    • Add these Anchors to the panel gameObject, namely completely stretched (required to make it work)

    • Add the component SUSafeAreaResizer to the panel object (required to make it work)


    Now our panel will be automatically resized to fit inside the Safe Area; in fact, if we change the "Game Window" size or if we go to the "Device Simulator" window we can see that our button is aligned to the left but it is fitting inside the Safe Area.
